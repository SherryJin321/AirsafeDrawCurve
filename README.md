# AirsafeDrawCurve

## 1. 软件名称<br>
   曲线绘制<br>
   
## 2. 软件版本<br>
### 2.1 版本 V1.0 <br>
#### 2.1.1 软件类型<br>
Visual Studio WPF应用程序<br>
#### 2.1.2 软件功能<br>
   * 通过浏览本机文件，选择所需的Excel文件。<br>
   * 软件导入Excel文件，将数据保存至DataTable变量中。<br>
   * 将DataTable变量中的数据，绘制成曲线，显示在DynamicDataDisplay控件中。<br>

已完成软件编程。<br>
